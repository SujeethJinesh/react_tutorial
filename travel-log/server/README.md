# Log entry

- Title - Text
- Description - Text
- Comments - Text
- Rating - scale of 1 - 10
- Image - Text - URL
- Start Date - DateTime
- End Date - DateTime
- Latitude - Number
- Longitude - Number
- Created At - Datetime
- Updated At - Datetime

In order to get the mongodb database working, you need to start the service with

`brew services start mongodb-community@4.2`

and stop it with

`brew services stop mongodb-community@4.2`
